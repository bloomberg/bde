Detailed Description

When used within the methods of Bind_TestTypeAlloc, the VALUE type will be const bslma::Allocator* and will keep track of which allocator is used with which argument of the Bind object. The sole purpose of deriving a new class is to use pretty printing of the allocators in verifySlots.
