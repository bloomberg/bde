Detailed Description

This class implements a value-semantic collection of unique predicates. (Note that a predicate is a compound entity that, as a whole, must be unique although individual parts need not be.) Additionally, the evaluate accessor can be used to determine if every predicate in the set evaluates to true in the context of a given attribute map.
