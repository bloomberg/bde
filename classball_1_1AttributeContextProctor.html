Detailed Description

This class implements a proctor that, on its own destruction, will destroy the attribute context of the current thread. Attribute contexts are stored in thread local memory. On destruction, objects of this type will deallocate the current threads attribute context (if one has been created), and set the thread local storage pointer to 0.
