Detailed Description

template<typename TYPE>
struct bsl::add_cv< TYPE >

This struct template implements the add_cv meta-function defined in the C++11 standard [meta.trans.cv] to provide a typedef type. If the (template parameter) TYPE is not a reference, nor a function, nor already const-qualified and volatile-qualified, then type is an alias to the same type as TYPE except that the top-level const-qualifier and volatile-qualifier has been added; otherwise, type is an alias to TYPE.
