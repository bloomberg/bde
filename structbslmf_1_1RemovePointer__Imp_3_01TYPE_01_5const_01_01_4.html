Detailed Description

template<class TYPE>
struct bslmf::RemovePointer_Imp< TYPE *const >

This partial specialization of RemovePointer_Imp, for when the (template parameter) TYPE is a const-qualified pointer type, provides an alias Type that refers to the type pointed to by TYPE. Note that this is specifically for const-qualified pointers, and not for pointers-to-'const'-type.
