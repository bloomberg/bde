Detailed Description

template<class PROTOTYPE>
struct bslmf::FunctionPointerTraits< PROTOTYPE *volatile >

This class gives information about the specified PROTOTYPE. The general definition gives no information, but specializations for function pointers types define nested types ResultType, ArgumentList, and Linkage.
