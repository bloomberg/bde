Detailed Description

template<typename T>
class bsl::reference_wrapper< T >

This class is a wrapper that encapsulates an object reference, enabling operations not possible on actual references, including assignment, copying, and storage in standard containers. When stored in a container, it enables functions defined to operate on references to the type represented to be called on the container elements.
