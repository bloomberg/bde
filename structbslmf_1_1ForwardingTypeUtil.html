template<class TYPE >
static TargetType bslmf::ForwardingTypeUtil< TYPE >::forwardToTarget ( typename ForwardingType< TYPE >::Type  v  )  [static]

Return (for the specified v parameter) std::forward<TYPE>(v), where v is assumed to originally have been an argument of TYPE after forwarding through an intermediate call chain. Specifically, if TYPE is an rvalue type, return an rvalue reference to v, otherwise return v unchanged, thus converting an rvalue copy into an rvalue move when possible. For compilers that do not support rvalue references, return v unchanged. This function is intended to be called to forward an argument to the final target function of a forwarding call chain. Note that this function is not intended for use with TYPE parameters of volatile-qualified rvalue type, which are effectively unheard of in real code and have strange and hard-to-understand rules.
