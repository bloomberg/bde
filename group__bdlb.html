Purpose:
Provide utilities on basic types.
MNEMONIC: Basic Development Library Basics (bdlb):
Description:
The bdlb package provides utility structs in the broad areas of bit manipulation, string handling, random number generations, hashing, variant and nullable types. Classes are provided for nullable and variant values.
