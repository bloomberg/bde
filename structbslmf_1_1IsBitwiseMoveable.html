Detailed Description

template<typename TYPE>
struct bslmf::IsBitwiseMoveable< TYPE >

Trait metafunction that determines whether the specified parameter TYPE is bit-wise moveable. If IsBitwiseMoveable<TYPE> is derived from bsl::true_type then TYPE is bit-wise moveable. Otherwise, bit-wise moveability cannot be inferred for TYPE. This trait can be associated with a bit-wise moveable user-defined class by specializing this class or by using the BSLMF_DECLARE_NESTED_TRAIT macro.
