Detailed Description

template<class TYPE>
class bdlb::TestInputIterator< TYPE >

Provide an input iterator that iterates over an empty sequence. All TestInputIterator objects compare equal. Since the iteration sequence is empty, incrementing or de-referencing this iterator yields undefined behavior.
