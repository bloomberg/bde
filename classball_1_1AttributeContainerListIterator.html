Detailed Description

This class provides a stl-style iterator over a sequence of AttributeContainer object addresses. The behavior of the operator* method is undefined unless the iterator is at a valid position in the sequence of AttributeContainer object addresses (i.e., not the "end") and the referenced element has not been removed since the iterator was constructed.
