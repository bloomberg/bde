 bdlar.txt

@PURPOSE: Provide dynamic dispatch types to manipulate attribute types

@MNEMONIC: Basic Development Library Attribute types Runtime (bdlar)

@DESCRIPTION: The `bdlar` package provides a runtime wrappers for `bdlat`.
