 bdl+inteldfp.txt

@PURPOSE: Provide third-party decimal floating point library (Intel DFP).

@MNEMONIC: Basic Development Library INTEL DFP (bdl+inteldfp)

@SEE_ALSO: bdldfp_decimal

@DESCRIPTION: The 'bdl+inteldfp' non-standard package contains the open-source
 Intel DFP library.  'bdl+inteldfp' should *not* be used directly by client
 code.  It is intended to be used (internally) *only* by 'bdldfp_decimal' and
 'bdldfp_decimalutil'.
