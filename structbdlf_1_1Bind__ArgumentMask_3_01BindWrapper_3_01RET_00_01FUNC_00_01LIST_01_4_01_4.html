Detailed Description

template<class RET, class FUNC, class LIST>
struct bdlf::Bind_ArgumentMask< BindWrapper< RET, FUNC, LIST > >

This specialization of Bind_ArgumentMask defines a mask for a BindWrapper object passed recursively as a bound argument. The value is not important, as long as it is out of range. This makes sure that a binder with a nested BindWrapper object is treated as non-explicit, in the same way as if the the nested binder was of type Bind.
