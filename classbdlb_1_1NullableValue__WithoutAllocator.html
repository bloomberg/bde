Detailed Description

template<class TYPE>
class bdlb::NullableValue_WithoutAllocator< TYPE >

This template class provides the implementation of NullableValue for types that do not require an allocator.

Assign to this nullable object the value of the (template parameter) TYPE (which does not have the bslma::UsesBslmaAllocator trait) created in place using the specified args.
