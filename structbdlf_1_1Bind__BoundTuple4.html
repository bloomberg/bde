Detailed Description

template<class A1, class A2, class A3, class A4>
struct bdlf::Bind_BoundTuple4< A1, A2, A3, A4 >

This struct stores a list of four arguments. It does not use the const-forwarding type of its arguments, unlike Bind_Tuple4 which applies that optimization to avoid unnecessary copying.
