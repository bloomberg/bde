Detailed Description

template<class KEY, class VALUE, class COMPARATOR = std::less<KEY>, class ALLOCATOR = allocator<bsl::pair<const KEY, VALUE> >>
class bsl::map< KEY, VALUE, COMPARATOR, ALLOCATOR >::value_compare

This nested class defines a mechanism for comparing two objects of value_type using the (template parameter) type COMPARATOR. Note that this class exactly matches its definition in the C++11 standard [23.4.4.1]; otherwise we would have implemented it as a separate component-local class.
