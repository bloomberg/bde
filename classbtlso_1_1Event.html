Detailed Description

This attribute class represents an event on a socket. The recognized events are btlso::EventType::e_ACCEPT, btlso::EventType::e_CONNECT, btlso::EventType::e_READ, and btlso::EventType::e_WRITE. Accessors and manipulators are provided for the socket handle and the event type.
