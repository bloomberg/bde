Description:
This component provides a single traits class, bslalg::TypeTraitBitwiseMoveable. An object of a TYPE that has the bit-wise moveable trait can be copied either by invoking the move constructor or by moving the footprint (i.e., the sizeof(TYPE) bytes at the object address) using memcpy.
This component is used by various bslalg components for providing optimized primitives for types that have the bit-wise moveable trait. The major benefit of this trait is not for a single object but for an array of such types, as a loop can be replaced by a single call to memcpy.
