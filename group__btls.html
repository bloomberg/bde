Purpose:
Provide transport-specific system level services.
MNEMONIC: Basic Transport Environment System-level services (btls):
Description:
The btls package provides "system level" services that are required by this transport package group in a form that is portable across all suported platforms. Currently, the package supplies objects and utilities to implement the Unix iovec ("scatter/gather") buffer structure in a platform-neutral form.
