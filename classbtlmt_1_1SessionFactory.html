Detailed Description

This class defines a pure protocol for the allocation and deallocation of Session objects.

Member Typedef Documentation

typedef bsl::function<void(int, Session*) btlmt::SessionFactory::Callback)

The signature of a function to be called after a new session is allocated. The first argument is taken to indicate the status of the allocation - 0 if successful, nonzero otherwise. The second argument is a pointer to the newly allocated session.
