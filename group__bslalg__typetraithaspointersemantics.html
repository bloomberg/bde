See also:
bslmf_typetraits
Description:
This component provides a single traits class, bslalg::TypeTraitHasPointerSemantics. A type with this trait is said to have "pointer semantics". That is the type behaves as if it were a fundamental pointer type. The type must define(at a minimum) operator* and operator->.
