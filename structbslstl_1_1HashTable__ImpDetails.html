Detailed Description

This utility struct provides a namespace for functions on iterators that are useful when implementing a hash table.

Member Function Documentation

static size_t bslstl::HashTable_ImpDetails::nextPrime ( size_t  n  )  [static]

Return the next prime number greater-than or equal to the specified n in the increasing sequence of primes chosen to disperse hash codes across buckets as uniformly as possible. Note that, typically, prime numbers in the sequence have increasing values that reflect a growth factor (e.g., each value in the sequence may be, approximately, two times the preceding value)
