Detailed Description

This struct provides a test class capable of holding up to 14 bound parameters of types TestArgNoAlloc[1--14], with full (non-streamable) value semantics defined by the operator==. By default, a Bind_TestTypeNoAlloc is constructed with nil (N1) values, but instances can be constructed with actual values (e.g., for creating expected values). A Bind_TestTypeNoAlloc can be invoked with up to 14 parameters, via member functions testFunc[1--14]. These functions are also called by the overloaded member operator() of the same signatures, and similar global functions testFunc[1--14]. All invocations support the above Bind_TestSlotsNoAlloc mechanism.

This struct intentionally does not take an allocator.
