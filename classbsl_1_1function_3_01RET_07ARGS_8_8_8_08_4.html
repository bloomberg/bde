Detailed Description

template<class RET, class... ARGS>
class bsl::function< RET(ARGS...)>

An instantiation of this class template generalizes the notion of a pointer to a function taking the specified ARGS types and returning the specified RET type, i.e., a function pointer of type RET(*)(ARGS). An object of this class wraps a run-time invokable object specified at construction, such as a function pointer, member function pointer, or functor. Note that function is defined only for template parameters that specify a function prototype; the primary template (taking an arbitrary template parameter) is not defined.
