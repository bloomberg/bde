Detailed Description

template<class VALUE>
struct bdlf::Bind_TestSlotsBase< VALUE >

This struct defines an array of VALUE to keep track (in conjunction with the Bind_TestTypeNoAlloc or Bind_TestTypeAlloc classes) of which value or allocator is being passed to which argument of the test function by a bdlf_bind object.
