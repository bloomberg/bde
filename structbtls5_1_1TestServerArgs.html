Constructor & Destructor Documentation

btls5::TestServerArgs::TestServerArgs ( bslma::Allocator *  basicAllocator = 0  )  [explicit]

Create a TestServerArgs object initialized as follows:

  • d_mode = e_SUCCEED_AND_CLOSE
  • d_reply = 0
  • d_label ""
  • d_verbosity = e_DEBUG
  • d_logStream_p = &bsl::cout
  • d_expectedIp = 0'
  • d_expectedPort = 0
  • d_expectedDestination default
  • d_expectedCredentials default

Optionally specify a basicAllocator used to supply memory. If basicAllocator is 0, the currently installed default allocator is used.
