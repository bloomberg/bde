Detailed Description

template<int ID>
class bdlf::Bind_TestArgAlloc< ID >

This class is used to disambiguate types in passing parameters due to the fact that Bind_TestArgAlloc<ID1> is a different type than Bind_TestArgAlloc<ID2> is ID1 != ID2. This class is used for testing memory allocator issues.
