static int bdls::PipeUtil::makeCanonicalName ( bsl::string *  pipeName,
const bslstl::StringRef &  baseName 
) [static]

Load into the specified pipeName the system-dependent canonical pipe name corresponding to the specified baseName. On Unix systems, attempts to load the environment variable TMPDIR to create the pipe name. Return 0 on success, and a nonzero value if baseName cannot be part of a pipe name on this system.
