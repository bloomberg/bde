Purpose:
Provide and time-related vocabulary types, and related utilities.
MNEMONIC: Basic Development Library Time (bdlt):
Author:
Steven Breitstein (sbreitstein)
Description:
The bdlt ("Basic Development Library Time") package provides vocabulary types for representing date and type values, utilities providing non-primitive functionality on those types, utilities that provide date- and time values (e.g., bdlt_currenttime), and supporting components.
