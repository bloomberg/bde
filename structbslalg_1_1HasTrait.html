Detailed Description

template<typename TYPE, typename TRAIT>
struct bslalg::HasTrait< TYPE, TRAIT >

This meta-function evaluates to bslmf::MetaInt<1> if the parameterized type TYPE has the parameterized TRAIT, and to bslmf::MetaInt<0> otherwise.
