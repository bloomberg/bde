This template declares a typelist of 0-20 types. For each type specified, a corresponding type will be declared as "Type<N>" where where N is the Nth parameter to this typelist(relative to 1). Additionally, the enumeration LENGTH is declared with a value equal to the length of this typelist. Each typelist also declares a member template TypeOf such that TypeOf<N>Type evaluates to the type of the Nth the in this typelist. The the type Type is also declared a length specific version of this typelist.
