Description:
This component defines a meta-function, bsl::remove_const, that may be used to remove the top-level const-qualifier from a type.
bsl::remove_const meets the requirements of the remove_const template defined in the C++11 standard [meta.trans.cv].
Usage:
In this section we show intended use of this component.
Example 1: Removing The Const-Qualifier of A Type:
Suppose that we want to strip the const-qualifier from a particular type.
First, we create two typedefs -- a const-qualified type (MyConstType) and the same type without the const-qualifier (MyType): 

  typedef int       MyType;
  typedef const int MyConstType;


Now, we strip the the const-qualifier from MyConstType using bsl::remove_const and verify that the resulting type is the same as MyType: 

  assert(true ==
        (bsl::is_same<bsl::remove_const<MyConstType>::type, MyType>::value));
