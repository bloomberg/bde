Detailed Description

This struct provides a namespace to implement the swap member function of deque<VALUE_TYPE, ALLOCATOR>. This function can be implemented irrespective of the VALUE_TYPE or ALLOCATOR template parameters which is why we implement it in this non-templated, non-inlined utility.
