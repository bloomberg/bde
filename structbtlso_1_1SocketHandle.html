Member Enumeration Documentation

anonymous enum

These codes represent a non platform specific error classifications. More than one platform specific code may map onto a single error classification.

Enumerator:
e_ERROR_EOF 

The peer closed its send side of the connection. This includes the normal closure of a connection but does not include when connection is terminated by the peer without a normal closure.

e_ERROR_UNCLASSIFIED 

The platform specific error code could not be mapped onto any error classification.

e_ERROR_CONNDEAD 

The connection does not represent a connected socket.

e_ERROR_WOULDBLOCK 

The socket is a non blocking socket and the call would block.

e_ERROR_NORESOURCES 

Resources were not available to complete the call.

e_ERROR_INTERRUPTED 

The call was interrupted. For receive and send calls no data was transferred.

e_ERROR_TIMEDOUT 

A system call timed out.
