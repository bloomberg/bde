Detailed Description

This complex-constrained (value-semantic) type describes the username and password for SOCKS5 authentication per RFC 1929.

Create a Credentials object having the specified username and password. Optionally specify a basicAllocator used to supply memory. If basicAllocator is 0, the currently installed default allocator is used.
