template<class TRUE_TYPE, class FALSE_TYPE>
struct bsl::conditional< false, TRUE_TYPE, FALSE_TYPE >

This partial specialization of bsl::conditional, for when the (template parameter) value COND is false, provides a typedef type that is an alias to the (template parameter) type FALSE_TYPE.
