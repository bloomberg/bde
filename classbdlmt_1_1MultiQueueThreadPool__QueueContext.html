Detailed Description

This private class encapsulates a lightweight job queue and a callback which processes the queue.

Member Typedef Documentation

typedef bsl::function<void()> bdlmt::MultiQueueThreadPool_QueueContext::QueueProcessorCb

This type defines a callback functor that is enqueued to the thread pool, and used to process elements of a specific queue. The queue ID must be bound to the functor at the time of its instantiation.
