Detailed Description

template<class T>
struct bsls::ProtocolTest_IsAbstract< T >

This class template is a compile-time meta-function, parameterized with type T, the output of which is VALUE, which will be true if T is abstract and false otherwise. The IsAbstract test makes use of the fact that a type an array of objects of an abstract type (e.g., U (*)[1]) cannot exist. Note that it is only an approximation, because this is also true for an incomplete type. But this approximation is good enough for the purpose of testing protocol classes.
