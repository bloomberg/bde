Detailed Description

template<class TYPE, template< class T > class TRAIT>
class bslmf::DetectNestedTrait_Imp< TYPE, TRAIT >

Implementation of class to detect whether the specified TRAIT parameter is associated with the specified TYPE parameter using the nested type trait mechanism. The VALUE constant will be true iff TYPE is convertible to NestedTraitDeclaration<TYPE, TRAIT>.
