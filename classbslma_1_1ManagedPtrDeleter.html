Detailed Description

This complex constrained in-core value-semantic class holds the information necessary for ManagedPtr correctly manage its underlying object, namely the addresses of object and factory, and the deleter function, optionally supplied through the constructors or through the set method. It is stored in a sub-structure to allow the compiler to copy it more efficiently.
