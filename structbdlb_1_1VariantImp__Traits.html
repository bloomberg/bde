Detailed Description

template<class TYPES>
struct bdlb::VariantImp_Traits< TYPES >

This struct is component-private. Do not use. This meta-function selects VariantImp_AllocatorBase as a base class type if any one of the twenty types held by a variant has the bslma::UsesBslmaAllocator trait, and VariantImp_NonAllocatorBase otherwise.
