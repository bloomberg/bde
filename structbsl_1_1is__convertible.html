Detailed Description

template<class FROM_TYPE, class TO_TYPE>
struct bsl::is_convertible< FROM_TYPE, TO_TYPE >

This struct template implements the is_convertible_dispatch meta-function defined in the C++11 standard [meta.rel] to determine if the (template parameter) FROM_TYPE is convertible to the (template parameter) TO_TYPE. This struct derives from bsl::true_type if the FROM_TYPE is convertible to TO_TYPE, and from bsl::false_type otherwise. Note that both FROM_TYPE and TO_TYPE should be complete types, arrays of unknown bound, or (possibly cv-qualified) void types.
