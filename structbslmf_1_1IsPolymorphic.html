Detailed Description

template<typename TYPE>
struct bslmf::IsPolymorphic< TYPE >

This struct template implements a meta-function to determine if the (template parameter) TYPE is a (possiblly cv-qualified) polymorphic type. This struct derives from bslmf::MetaInt<1> if the TYPE is a polymorphic type, and bslmf::MetaInt<0> otherwise.

Note that although this struct is functionally equivalent to bsl::is_polymorphic, the use of bsl::is_polymorphic should be preferred.
