Detailed Description

This struct should be used by only shared_ptr constructors. Its purpose is to enable shared_ptr constructors to determine if the (template parameter) types COMPATIBLE_TYPE or ELEMENT_TYPE have a specialization of enable_shared_from_this as an unambiguous, publicly accessible, base class.

If the data member d_weakThis of the specified result has expired, then set it to refer to the object managed by the specified sharedPtr; otherwise (i.e., d_weakThis has not expired) this method has no effect. This function shall be called only by shared_ptr constructors creating shared pointers for classes that derive publicly and unambiguously from a specialization of enabled_shared_from_this. Note that overload resolution will select the overload below if a supplied type does not derive from a specialization of enable_shared_from_this.
