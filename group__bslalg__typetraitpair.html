Component bslalg_typetraits
Description:
This component provides a single traits class, bslalg::TypeTraitPair. A TYPE that has this trait fulfills the following requirements, where mX is a modifiable object and X a non-modifiable object of TYPE: 

  Valid expression     Type
  ----------------     ----
  TYPE::first_type
  TYPE::second_type

  mX.first             first_type
  mX.second            second_type
  X.first              const first_type
  X.second             const second_type


Note that first and second are not member functions, but data members.
