static unsigned bdldfp::DenselyPackedDecimalImpUtil::encodeDeclet ( unsigned  digits  )  [static]

Return an unsigned integer whose bits encode the value of the specified digits in the Densely Packed Decimal (DPD) format. The behavior is undefined unless 0 <= digits < 1000. Note that the result will be a bit pattern whose binary representation corresponds to a value in the range [0, 1024]. Also note that this function encodes a single declet (not a complete DPD decimal floating point representation), and returns an unsigned int as it is intended for use with raw bit manipulation functions.

static unsigned bdldfp::DenselyPackedDecimalImpUtil::decodeDeclet ( unsigned  declet  )  [static]

Return the (unsigned integer) value encoded in the specified declet. The behavior is undefined unless the binary value of declet < 1024, declet is encoded in Densely Packed Decimal (DPD), and every unused bit in the densely packed state is set to 0. Note that the result will be an integer in the range [0, 999]. Also note that this function decodes a single declet (not a complete DPD decimal floating point representation), and returns unsigned int type values, as it is intended for use with raw bit manipulation functions.

Return a StorageType32 object representing a decimal floating point number consisting of the specified significand and exponent, with the sign given by the significand. If significand is 0, the result is 0 but the quanta of the result is unspecified. The behavior is undefined unless abs(significand) <= 9,999,999 and -101 <= exponent <= 90.
