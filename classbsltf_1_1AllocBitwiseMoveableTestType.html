Detailed Description

This unconstrained (value-semantic) attribute class that is bitwise-moveable, uses a bslma::Allocator to allocate memory, and defines the type traits bslma::UsesBslmaAllocator and bslmf::IsBitwiseMoveable. See the Attributes section under @DESCRIPTION in the component-level documentation for information on the class attributes.
