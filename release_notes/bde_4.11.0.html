BDE 4.11.0 Release¶

Schedule¶

  • The BDE team announces that the BDE 4.11.0 production release was completed on Wednesday, July 17, 2024.

BDE 4.11.0 Release Highlights¶

This release consists of assorted minor fixes.

Fixed DRQSs:¶

Summary

Local-to-UTC methods of ‘baltzo::TimeZoneUtilImp’ do not return error on ‘Datetime’ overflow

complete bdlm_metricsregistry test driver

Clean up warning: unknown warning group in Clang

bbs_build: add support for .g.cpp files

bsl::string_view, some C++14 constexpr members are not legitimately constexpr

BER decoding crash on bogus string length

UBSAN: signed integer overflow in bdlcc_timequeue.h

[sun][c++03] compile errors for googlemock mock objects when bsl_variant.h is included

is_invocable_r_v returns size_t instead of bool

bslmf::MovableRef documentation contains misleading usage for move constructors

bdlmt_signaler: TC 4 nightly build failure on AIX

balber_berdecoder: Fuzz test crash

Add preliminary MSVC on ARM support

please address bdlmt_eventscheduler TC 1 intermittant failure

reduce using perl bde clang-15 NB warnings

correct residual !… clang warnings in bde

reduce clang-15 bde warnings some more

please correct bdlm_metricsregistry.t runtime failures on AIX

Please fix bslmf_invokeresult nightly build failure

balber_berdecoder: Fuzz test crash 2

bdlma_localbufferedobject_cpp03 inconsistency

reduce bal nighlty build warnings

reduce bal clang-15 test driver compile warnings
