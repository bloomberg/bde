Detailed Description

This component-private class is a concrete implementation of the AttributeContainer protocol for a single attribute.

Create a BALL attribute container holding a single rule, associating the specified name with the specified value. Optionally specify a basicAllocator used to supply memory. If basicAllocator is 0 or unspecified, the currently installed default allocator is used.
