Detailed Description

template<class TYPE, TYPE VAL>
struct bsl::integral_constant< TYPE, VAL >

Generate a unique type for the given TYPE and VAL. This struct is used for compile-time dispatch of overloaded functions and as the base class for many metafunctions.
