Detailed Description

Provides a scoped guard that sets BALL attributes for "serviceId" and "uuid" in the current thread.

Set BALL logging attributes for the current thread for the scope of this object, associating the specified name with the specified value. Optionally specify a basicAllocator used to supply memory. If basicAllocator is 0 or unspecified, the currently installed default allocator is used.
