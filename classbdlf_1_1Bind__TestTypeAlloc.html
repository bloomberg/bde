Detailed Description

This class provides a test class capable of holding up to 14 bound parameters of types Bind_TestArgAlloc[1--14], with full (non-streamable) value semantics defined by the operator==. By default, a Bind_TestTypeAlloc is constructed with nil (k_N1) values, but objects can be constructed with actual values (e.g., for creating expected values). A Bind_TestTypeAlloc can be invoked with up to 14 parameters, via member functions testFunc[1--14]. These functions are also called by the overloaded member operator() of the same signatures, and similar global functions testFunc1--14. All invocations support the above Bind_TestSlotsAlloc mechanism.

This class intentionally does take an allocator.
