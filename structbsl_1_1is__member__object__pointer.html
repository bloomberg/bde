Detailed Description

template<class TYPE>
struct bsl::is_member_object_pointer< TYPE >

This struct template implements the is_member_object_pointer meta-function defined in the C++11 standard [meta.unary.cat] to determine if the (template parameter) TYPE is a pointer to non-static data member type. This struct derives from bsl::true_type if the TYPE is a pointer to non-static data member type, and from bsl::false_type otherwise.
