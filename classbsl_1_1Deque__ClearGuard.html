template<class VALUE_TYPE, class ALLOCATOR>
class bsl::Deque_ClearGuard< VALUE_TYPE, ALLOCATOR >

This class provides a proctor which, at destruction, calls clear on the deque passed at construction, unless the guard has been released prior.
