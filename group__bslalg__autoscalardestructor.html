See also:
Component bslma_autodestructor, Component bslalg_scalardestructionprimitives
Description:
This component provides a proctor object to manage an otherwise-unmanaged instance of a user-defined type. If not explicitly released, the instance managed by the proctor object is automatically destroyed by the proctor's destructor, using the bslalg::ScalarDestructionPrimitives utility.
