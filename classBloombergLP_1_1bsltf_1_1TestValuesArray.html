Detailed Description

template<class VALUE, class CONVERTER = TestValuesArray_DefaultConverter<VALUE>>
class BloombergLP::bsltf::TestValuesArray< VALUE, CONVERTER >

This class provide a container to store values of the parameterized VALUE, and also provide the iterators to access the values. The iterators are designed to conform to a standard input iterator, and will report any misuse of the iterator.
