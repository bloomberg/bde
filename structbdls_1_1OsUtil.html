static int bdls::OsUtil::getOsInfo ( bsl::string *  osName,
bsl::string *  osVersion,
bsl::string *  osPatch 
) [static]

Load the operating system name, version name and patch name into the specified osName, osVersion and osPatch respectively. Return 0 on success and a non-zero value otherwise. On MS Windows systems osName will be Windows, with osVersion set to dwMajorVersion concatenated with a dot and dwMinorVersion, and osPatch will be set to szCSDVresion (Service Pack name), as returned by GetVersionEx. On Posix systems, osName, osVersion and osPatch will be set to sysname, release, and version, respectively, as returned by uname(2).
