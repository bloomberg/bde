int btlmt::TcpTimerEventManager_ControlChannel::clientWrite ( bool  forceWrite = false  ) 

Write the control byte into the client handle. If the specified forceWrite is true write to the client handle even if a previous write is outstanding. By default, forceWrite is false and the control byte is not written to the client handle if a previous write is outstanding. Return the number of bytes written on success, and a negative value otherwise. Note that forceWrite can be used when a write might be lost because the control channel had closed and ensures that the control byte is unconditionally written to the client handle.
