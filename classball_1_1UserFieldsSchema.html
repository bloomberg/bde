Detailed Description

This class implements a value-semantic type for describing the contents of a ball::UserFields object. A ball::UserFieldSchema object maintains a sequence of field names and field data types. The name and type at a given index in a schema object indicate the name of a field and its data type in the ball::UserFields object being described. Additionally, a schema object provides the method indexOf to lookup the index of a field given its name.
