Detailed Description

This class provides a protocol for determining values based on dates according to derived implementations of specific day-count conventions.

Return the (signed fractional) number of years between the specified beginDate and endDate. If beginDate <= endDate, then the result is non-negative. Note that reversing the order of beginDate and endDate negates the result; specifically, |yearsDiff(b, e) + yearsDiff(e, b)| <= 1.0e-15 for all dates b and e.
