bslma::Allocator *  basicAllocator = 0 
) [static]

Return a shared pointer with an in-place representation holding a newly-created uninitialized buffer of the specified bufferSize (in bytes). Optionally specify a basicAllocator used to supply memory. If basicAllocator is 0, the currently installed default allocator is used.
