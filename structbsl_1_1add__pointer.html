Detailed Description

template<class TYPE>
struct bsl::add_pointer< TYPE >

This struct template implements the add_pointer meta-function defined in the C++11 standard [meta.trans.ptr], providing an alias, type, that returns the result. If the (template parameter) TYPE is not a reference type, then type is an alias to a pointer type that points to TYPE; otherwise, type is an alias to a pointer type that points to the type referred to by the reference TYPE.
