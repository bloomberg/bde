Detailed Description

template<class TYPE, template< class T > class TRAIT, bool COND = true>
class bslmf::NestedTraitDeclaration< TYPE, TRAIT, COND >

Class TYPE will be convertible to NestedTraitDeclaration<TYPE,TRAIT,true> if TRAIT is associated with TYPE using the BSLMF_NESTED_TRAIT_DECLARATION macro. Nested trait detection depends on COND being true. If COND is false, the nested trait detection will not see the conversion it is looking for and will not associate TRAIT with TYPE. This feature is used by BSLMF_NESTED_TRAIT_DECLARATION_IF to turn a trait on or off depending on a compile-time condition (usually another trait).
