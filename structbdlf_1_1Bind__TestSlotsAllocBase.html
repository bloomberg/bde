Detailed Description

template<class AllocPtr>
struct bdlf::Bind_TestSlotsAllocBase< AllocPtr >

The point of this base class is that we would like it to be a class template, so that the class data does not pollute the library object files. Since there is no reason to make Bind_TestSlotsAlloc a template, we separate the static pointers into this base class. These pointers are needed for the pretty printing in the verifySlots method.
