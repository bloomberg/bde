Detailed Description

Provide read-only sequential access to the allocated sessions in a SessionPool object.

Constructor & Destructor Documentation

btlmt::SessionPoolSessionIterator::SessionPoolSessionIterator ( SessionPool *  sessionPool  )  [explicit]

Create an iterator for the specified sessionPool and associate it If the catalog is empty than the iterator is initialized to be invalid. The catalog is read locked for the duration of iterator's life.

Member Function Documentation

void btlmt::SessionPoolSessionIterator::operator++ (  ) 

Advance this iterator to refer to the next object of the associated session pool; if there is no next object in the associated session pool, then this iterator becomes invalid. The behavior is undefined unless this iterator is valid. Note that the order of the iteration is implementation-defined.
