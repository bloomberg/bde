Component bslmf_addvolatile
Description:
This component defines a meta-function, bsl::remove_volatile, that may be used to remove the top-level volatile-qualifier from a type.
bsl::remove_volatile meets the requirements of the remove_volatile template defined in the C++11 standard [meta.trans.cv].
Usage:
In this section we show intended use of this component.
Example 1: Removing The Volatile-Qualifier of A Type:
Suppose that we want to strip the volatile-qualifier from a particular type.
First, we create two typedefs -- a volatile-qualified type (MyVolatileType) and the same type without the volatile-qualifier (MyType): 

  typedef int          MyType;
  typedef volatile int MyVolatileType;


Now, we strip the the volatile-qualifier from MyVolatileType using bsl::remove_volatile and verify that the resulting type is the same as MyType: 

  assert(true ==
  (bsl::is_same<bsl::remove_volatile<MyVolatileType>::type, MyType>::value));
