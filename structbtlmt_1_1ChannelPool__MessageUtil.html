Detailed Description

This is an implementation type of ChannelPool and should not be used by clients of this component. The ChannelPool_MessageUtil struct provides a namespace for a set of common functions that can be applied to the different message container types supported by a ChannelPool.

Member Enumeration Documentation

anonymous enum

This enumeration defines the constant e_MAX_IOVEC_SIZE, which is used to indicate the maximum length of an array of iovecs that can be directly read from or written to a socket. Note that IOV_MAX is defined (on POSIX unix platforms) in "limits.h", and indicates the maximum number of iovecs that can be supplied to writev.
