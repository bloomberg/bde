Detailed Description

template<int ID>
class bdlf::Bind_TestArgNoAlloc< ID >

This very simple struct is used purely to disambiguate types in passing parameters due to the fact that Bind_TestArgNoAlloc<ID1> is a different type than Bind_TestArgNoAlloc<ID2> is ID1 != ID2. This class does not take an optional allocator.
