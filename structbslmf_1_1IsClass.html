bslmf::IsClass< TYPE > Struct Template Reference

#include <bslmf_isclass.h>

Detailed Description

template<class TYPE>
struct bslmf::IsClass< TYPE >

This meta-function derives from bsl::true_type if the (template parameter) TYPE is a class type, or a reference to a class type, and from bsl::false_type otherwise.

Note that although this struct is functionally identical to bsl::is_class, the use of bsl::is_class should be preferred.
