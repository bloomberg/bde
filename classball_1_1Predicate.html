Detailed Description

This class contains an attribute name and an attribute value, both of which are managed. A Predicate object contains a Attribute subobject, so that an equivalent Attribute object can be obtained without any overhead when needed. Since the Attribute subobject does not manage the attribute name, Predicate stores the attribute name in another member variable.
