Detailed Description

template<class ADDRESS>
class btlso::StreamSocketFactoryAutoDeallocateGuard< ADDRESS >

This class implements a proctor for a socket, using the specified factory to deallocate it upon destruction unless this socket has been released from management by this proctor prior to destruction.
