Return a size_t hash value corresponding to the specified key. Note that the return value is seemingly random (i.e., the hash is good) but identical on all platforms (irrespective of endianness).

NOTE: We reserve the right to change these hash functions to return different values. The current implementation only returns a 32 bit value -- when native_std::size_t is 64 bits, the high-order 32 bits of the return value are all zero. This is not a feature, it is a bug that we will fix in a later release.
